# LiveCampus GPA Calculator (Score-based)

LiveCampus Uで、成績情報詳細ページを開いた際に自動でGPAを計算して表示するChrome拡張機能。

## 特徴
- 取得単位数と点数をDOMから直接読み取り計算
- 履修中の科目や、点数が空欄の科目は自動で計算から除外
- 以下の独自のスコア基準でGPを算出:
  - 90点以上: 4.0
  - 85点以上: 3.5
  - 80点以上: 3.0
  - 75点以上: 2.5
  - 70点以上: 2.0
  - 65点以上: 1.5
  - 60点以上: 1.0
  - 60点未満: 0.0

## 使い方
1. 拡張機能がオンになっていることを確認する。
2. LiveCampus U にログインし、「成績情報」のページを開く。
3. 成績テーブルのすぐ上に、計算された「総合GPA」と「取得単位数」が表示される。
   *(※すでにページを開いている場合はリロードが必要)*

## 注意事項
- 本拡張機能は非公式のツールであり、大学公式のGPA算出結果を保証するものではありません。
- LiveCampus側のシステムアップデートによるHTML構造（DOM）の変更があった場合、正しく動作しなくなる可能性があります。
